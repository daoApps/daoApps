# 调试经验知识体系索引

## 概述

本索引文件用于指导团队成员访问和使用调试经验知识体系。

## 文档位置

所有调试经验文档存储在项目根目录的 `debugging-experience` 文件夹中：

```
d:\daoCollective\daoApps\daoApps\debugging-experience\
├── README.md              # 主文档
├── debugging_data.md      # 调试过程数据
├── debugging_experience.md # 调试经验与最佳实践
└── problem_solution_mapping.md # 问题-解决方案映射
```

## 访问方式

1. **本地访问**: 直接打开文件系统中的文档
2. **版本控制**: 文档已纳入Git版本控制
3. **团队共享**: 通过代码仓库共享给团队成员

## 检索方法

### 按问题类型检索
1. 打开 `problem_solution_mapping.md`
2. 查找对应的问题类型
3. 获取解决方案和经验参考

### 按技术领域检索
1. 打开 `debugging_experience.md`
2. 导航到相应的技术领域
3. 查看相关经验和最佳实践

### 按调试过程检索
1. 打开 `debugging_data.md`
2. 查看完整的调试时间线
3. 了解问题的具体解决过程
