# @agentpit/ui

AgentPit UI Component Library

## Installation

```bash
npm install @agentpit/ui
```

## Usage

```vue
<script setup>
import { Button } from '@agentpit/ui'
import '@agentpit/ui/styles'
</script>

<template>
  <Button>Hello World</Button>
</template>
```

## Documentation

Visit [AgentPit UI Documentation](https://ui.agentpit.com) for more details.
