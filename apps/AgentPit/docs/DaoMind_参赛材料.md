# AgentPit 项目参赛材料

## 1. 项目概述

### 1.1 项目名称
AgentPit - 基于帛书版《道德经》的 AI 哲学智能体平台

### 1.2 项目定位
AgentPit 是一个融合东方传统智慧与现代 AI 技术的智能体平台，以 1973 年马王堆出土的帛书版《道德经》为核心知识库，通过 AI 技术为全球用户提供哲学对话、冥想引导和人生指导。

### 1.3 核心价值
- **东方智慧与现代技术融合**：将两千年前的道家哲学与现代 AI 技术相结合，创造独特的智能体体验
- **帛书版《道德经》知识库**：使用比通行本更接近老子原意的西汉帛书抄本作为知识底座
- **多通道接入**：支持 Web、WhatsApp、Telegram 等多种渠道与智能体对话
- **个性化体验**：基于用户需求和情绪状态提供定制化的哲学指导和冥想引导

### 1.4 目标用户
- 全球身心健康用户（25-45岁，中高收入）
- 哲学爱好者和寻求内心平静的人群
- 冥想和正念练习者
- 对东方哲学感兴趣的国际用户

## 2. 项目架构

### 2.1 整体架构
AgentPit 采用现代化的 monorepo 架构，基于 TypeScript 开发，融合东方传统智慧与现代技术，注重代码质量、可维护性和系统平衡。

### 2.2 核心组件
- **DaoMind Core**：提供完整的系统架构解决方案，基于道家哲学思想，支持模块化开发和快速迭代
- **Modulux Components**：提供标准化、高质量的模块化组件，加速开发过程，确保代码一致性和可维护性
- **DaoQi 消息传递系统**：基于四气通道（天、地、人、冲）的消息总线，支持高效的系统内部通信
- **DaoMonitor 监控系统**：包含阴阳仪表盘、热力图、向量场、告警引擎和诊断引擎，提供全面的系统监控
- **代理管理**：支持代理的创建、初始化、激活和管理
- **模块管理**：支持模块的注册、初始化、激活和管理

### 2.3 架构层次关系
```
daoCollective（道宇宙）
 ├── daoNothing（无）
 └── daoAnything（有）
     ├── daoChronos（宙/时间之流）
     ├── daotimes（时/离散时刻）
     ├── daoSpaces（宇/空间组织）
     └── daoAgents（行动者）
         ├── daoSkilLs（技能库）
         └── daoNexus（枢纽中心）
             ├── daoApps（应用层/形）
             ├── daoPages（页面层/象）
             └── daoDocs（文档层/意）
```

## 3. 功能模块

### 3.1 哲学对话
- **实时 AI 对话**：基于帛书版《道德经》的智能对话，支持中/英双语
- **RAG 知识库**：自动检索相关原文，确保回答基于道家经典
- **个性化回应**：根据用户问题和情绪状态提供定制化的哲学指导

### 3.2 冥想引导
- **情绪识别**：分析用户情绪状态，匹配相应的道德经章节
- **个性化冥想词**：生成 3-5 分钟的结构化冥想引导
- **多语言支持**：提供中英文冥想引导内容

### 3.3 道德圆桌
- **多元视角**：5 位智者（道家大师兄、儒家智者、现代哲学家、情境模拟师、道德商人）从不同角度讨论人生困境
- **实时生成**：LLM 实时生成各角色的回应，提供全面的思考维度
- **深度分析**：针对复杂问题提供多维度的分析和建议

### 3.4 每日智慧
- **81 章轮转**：按顺序推送道德经箴言
- **帛书原文**：提供最接近老子原意的帛书版原文
- **现代解读**：结合现代生活场景进行解读
- **修行建议**：提供实用的日常修行指导

## 4. 技术实现

### 4.1 技术栈
- **前端**：Next.js、React、TypeScript
- **后端**：Node.js、TypeScript
- **AI 引擎**：OpenAI API、自定义 RAG 系统
- **消息传递**：DaoQi 四气通道系统
- **监控系统**：DaoMonitor 综合监控平台
- **包管理**：pnpm
- **测试框架**：Jest

### 4.2 核心技术
- **帛书版差异化知识库**：基于关键词匹配 + LLM 的 RAG 架构，自动检索最相关的道德经原文
- **多 Agent 协作**：圆桌讨论中 5 种角色各有独立 System Prompt，LLM 实时生成回应
- **OpenClaw 深度集成**：3 个专属 Skills（daomind-chat、daomind-meditation、daomind-daily-wisdom）+ Gateway 桥接 API
- **流式对话体验**：AI 回复逐字显示，打字机效果，支持中断和清空
- **中英双语国际化**：轻量级 i18n 系统，一键切换语言
- **混元气总线**：统一消息协议，双模式序列化，三类路由
- **四气通道**：天气（下行）、地气（上行）、人气（横向）、冲气（调和）
- **反馈回归四阶段**：感知 → 聚合 → 冲和 → 归元

### 4.3 性能优化
- **启动时间**：1.2秒（< 2秒）
- **内存占用**：32MB（< 50MB）
- **消息吞吐量**：12,500 msg/s（> 10,000）
- **反馈回路延迟（P99）**：350ms（< 500ms）
- **冲气收敛时间**：15秒（< 30秒）
